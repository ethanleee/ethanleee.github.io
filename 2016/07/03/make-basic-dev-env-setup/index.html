发表于 2016-07-03

Windows 下的虚拟机，虚拟机里面的是Ubuntu Mate 14.04 。

虚拟机virtualBOX

安装ssh-server

然后确认sshserver是否启动了：（或用“netstat -tlp”命令）

ps -e | grep ssh

如果只有ssh-agent那ssh-server还没有启动，需要

/etc/init.d/ssh start

如果看到sshd那说明ssh-server已经启动了。

ssh-server配置文件位于/ etc/ssh/sshd_config，在这里可以定义SSH的服务端口，默认端口是22，你可以自己定义成其他端口号，如222。然后重启SSH服务：

sudo /etc/init.d/ssh resart
#GSSAPIAuthentication yes
#GSSAPIDelegateCredentials no

然后重新启动 ssh 服务即可：

sudo /etc/init.d/ssh restart

Ruby

rvm install 2.3.1
rvm use 2.3.1

rvm 是 Ruby 的版本管理工具，其作用是在系统中安装若干个不同版本的 Ruby，且不让它们之间发生冲突。你可以安装很多个版本的 Ruby，比如，刚刚安装了 2.3.1，随后你还可以安装 1.9.2 ：

rvm install 1.9.2

#https://ruby.taobao.org/ ，因为直接安装实在痛苦，所以使用中国Ruby镜像。
sed -i -E 's!https?://cache.ruby-lang.org/pub/ruby!https://ruby.taobao.org/mirrors/ruby!' $rvm_path/config/db

rvm use 1.9.2 #需要使用 1.9.2 版本的 Ruby 的时候，就可以用这个命令：

rvm use 2.3.1 #也可以随时用这个命令切换回 2.3.1

rvm list # 你也随时可以用这个命令查看自己都安装过哪些版本：

rvm uninstall 1.9.2 #想要卸载哪个版本的 Ruby，你就用这样的命令：

nvm

Website : https://github.com/creationix/nvm
